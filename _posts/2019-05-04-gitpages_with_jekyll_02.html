걱정마라. 이미 jekyll은 plugin으로 admin을 제공해주고 있다.
생각이상으로 jekyll은 좋다.

설치하고 jekyll이 잘 설치가 되었다면 gem file 을 열어서
gem "jekyll", "~> 3.8.5”
해당 부분 주석 처리 후

아래코드를 저장시킨 후
gem "jekyll-admin", group: :jekyll_plugins

빌드 후 서버 재기동
bundle exec jekyll serve

그리고 아래페이지로 접속하면 page관리를 쉽게 해줄 admin 페이지가 보인다.

http://localhost:4000/admin/

별거 없어서 설명도 필요없을 것 같다.
