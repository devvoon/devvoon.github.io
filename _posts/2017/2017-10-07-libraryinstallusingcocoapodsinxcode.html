최근에, google api를 사용해 보려다가 필수적으로 알게된 cocoapods
google firebase, map api 등을 iOS에서 이용하고 싶다면,
반드시 설치해야 한다.

CocoaPods는 Xcode 프로젝트 개발시(Objective-C, Switf)
라이브러리를 손쉽게 설치할 수 있게
Ruby로 만들어졌으며, macOS환경이라면 사용가능하다.

Xcode에 CocoaPods를 이용하여 라이브러리 설치는
CocoaPods홈페이지에 자세히 안내되어있다.

 

1. terminal을 통해서 cocoapods를 설치

$ sudo gem install cocoapods

2. 프로젝트 경로로 이동후, pod를 설치

$ pod init

3. 프로젝트 경레에 Podfile이 생성되는데,
해당 podfile을 열고
추가해 줄 라이브러리를 아래와 같이 입력

pod 'Firebase/Core'

4. 터미널 창으로 다시 돌아와서 추가된 라이브러리 설치

$ pod install

5. 반드시 .xcworkspace 로 프로젝트 열어주기 열어주기

$ open your-project.xcworkspace
